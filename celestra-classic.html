Download

Use the $ sign into the Celestra with a small plugin.

Tested on desktop browsers (latest Chrome, latest Firefox, latest Edge, IE11) and mobile devices (iOS Safari, Chrome, Firefox and Android Browser, Chrome, Firefox and W10M Edge).

Latest version: 1.1.0

Minimal required Celestra version: 1.15.0

Development version: celestra-classic-1.1.0.js (337 byte)

Minimal version: celestra-classic-1.1.0.min.js (144 byte)

Functions

$(<selector>[,context]); Same as the qs(<selector>[,context]); function.
$$(<selector>[,context]); Same as the qsa(<selector>[,context]); function.
$A(<value>); Same as the toArray(<object>); function.
