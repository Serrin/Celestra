This is a Celestra demo plugin.

Tested on desktop browsers (latest Chrome, latest Firefox, latest Edge) and mobile devices (iOS Safari, Chrome, Firefox and Android Chrome, Samsung Internet, Firefox, Edge).

Latest version: 4.2.0

Download: celestra-demo-plugin.js

Source of the plugin


/**
 * @name Celestra demo plugin
 * @version 4.2.0
 * @see https://github.com/Serrin/Celestra
 * @license MIT https://opensource.org/licenses/MIT
 * Required Celestra version: 4.2.0
 */

(function(celestra){
"use strict";

celestra.sum = (...a) => a.slice(1).reduce((acc, v) => acc + v, a[0]);
celestra.avg = (...a) => a.slice(1).reduce((acc, v) => acc + v, a[0]) / a.length;

}(celestra));

Functions

Name Description
sum(); This function returns the sum value from the parameter values.
avg(); This function returns the average value from the parameter values.

Samples

sum();

alert( _.sum(3) );
alert( _.sum(3, 5) );
alert( _.sum(3.14, -5) );
alert( _.sum(true, 3.14, -9, 'Arthur Dent') );
alert( _.sum() );

avg();

alert( _.avg(2, 4, 3) );
alert( _.avg(2, 4, 3.14) );
alert( _.avg(2, -8, 3.14) );
